# LinguaGoWords - Telegram Mini App

Ushbu loyiha **LinguaGoWords** Telegram boti uchun maxsus tayyorlangan tezkor va interaktiv Mini App (Web App) interfeysidir. Loyiha toza HTML, CSS va JavaScript yordamida yaratilgan.

## 🚀 Xususiyatlari
- **Telegram Jildiga Moslashuvchanlik:** Telegram interfeysi ranglari va tungi/kunduzgi rejimlariga avtomatik moslashadi.
- **Tezkor Integratsiya:** `tg.sendData()` orqali botga qo'shimcha API'larsiz buyruqlar yuboradi.
- **Dinamik Statistika:** Botdan URL parametrlar orqali foydalanuvchi ballari va darajalarini qabul qila oladi.

## 🛠 Texnologiyalar
- HTML5
- CSS3 (Dinamik CSS o'zgaruvchilari bilan)
- JavaScript (Telegram WebApp SDK)

## 📦 Botga ulash tartibi
1. Ushbu loyihani biror xostingga yuklang (masalan, GitHub Pages, Vercel yoki Netlify).
2. Olingan `https://...` havolasini **@BotFather** orqali botingizning **Menu Button** yoki **Web App** bo'limiga joylang.
